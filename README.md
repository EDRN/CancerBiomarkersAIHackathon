# Cancer Biomarkers AI Hackathon
Serving files related to the hackathon for the Cancer Biomarkers AI workshop Aug 2024. The files are publicly available from EDRN (Lead PI: John Heine, data Custodian: Erin Fowler).

Hack-a-thon details: 
https://edrn.nci.nih.gov/news-and-events/upcoming-meetings/ai-bioinformatics-workshop/hackathon/

Use the notebook in the metadatasets subdir to explore the metadata associated with the three datasets to determine what problems can be tackled.

The README.pdf and the screening protocol doc file are common to all three sets as the three sets are related
